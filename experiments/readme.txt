Contents:
- main.py, main program. Run "python main.py --dataset=dataset_name", where dataset_name is one of the following dataset names (see Section 3 of "On Training Locally Adaptive CP" for more info), ['homes', 'CASP', 'concrete', 'energy', 'facebook_1', 'synth-cos', 'synth-squared', 'synth-inverse', 'synth-linear']
- functions.py, auxiliary functions. A file containing all auxiliary function required to run main.py
- datasets, folder containing all real-world data sets. 
- results, folder containing the output files.
